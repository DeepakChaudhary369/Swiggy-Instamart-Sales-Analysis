# Swiggy-Instamart-Sales-Analysis(Power BI)
# Project Overview

This project presents an interactive Power BI dashboard built to analyze retail sales data inspired by a Swiggy Instamartâ€“style quick commerce business. The goal is to transform raw transactional data into meaningful business insights that support data-driven decision-making.


# Objectives

Analyze overall sales and order performance

Compare performance across city tiers, shop sizes, and shop types

Identify top-performing product categories

Track shop expansion trends over the years

Present insights using clear and interactive visualizations

# Key KPIs

Total Sales

Total Orders

Average Sales Value

Average Customer Rating

# Key Insights

Tier 3 cities contribute the highest sales and order volume

Medium-sized shops perform more efficiently than small and large shops

Fruits & Vegetables and Snack Foods dominate overall sales

Customer satisfaction remains stable with an average rating of ~3.9

Shop openings decline after 2021, indicating a shift from expansion to optimization
